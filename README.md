# Hangman Game – Python Console Project

A simple and interactive text based Hangman game developed using Python as part of the CodeAlpha Python Programming Internship.

## Project Overview

This project is a console-based implementation of the classic Hangman game where the player guesses a hidden word one letter at a time. The game includes multiple Hangman stages, input validation, limited incorrect attempts, and dynamic word tracking through terminal interaction.

The project was built to strengthen fundamental Python programming concepts and improve logical thinking and problem-solving skills.

## Features

* Random word selection from a predefined list
* Letter by letter guessing system
* Interactive ASCII style Hangman stages
* Maximum of 6 incorrect attempts
* Input validation for invalid entries
* Win and loss condition handling
* Console-based user interaction

## Concepts Used

* Python `random` module
* `while` loops
* `if-else` conditional statements
* String manipulation
* List operations
* User input handling
* Console formatting
